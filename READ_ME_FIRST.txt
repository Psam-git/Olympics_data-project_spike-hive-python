This Project entailed the querying of Athletes Data from the Rio Games 2016 

Data Files -
The data folder countains Athletes, Diversity, Country_code (for linking diversity data) and funding data 

The Queries Folder Countains - results of the queries perfomed on the data set. Most of the data was queried in Hive using the cloudera hadoop platform. The funding data is transformed using spark.

Coding included should be opened in notepad++ before commencing use for easy use
